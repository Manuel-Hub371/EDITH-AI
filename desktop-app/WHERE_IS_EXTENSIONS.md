# 📦 Where to Find the Extensions Panel

**Quick Visual Guide for EDITH IDE**

---

## 🎯 3 Ways to Open Extensions

### Method 1: Activity Bar (Recommended)
```
Look at the LEFT SIDE of your IDE:

┌─────┐
│  📁 │ ← Explorer (files)
│  🔍 │ ← Search
│  🌿 │ ← Git/Source Control
│  📦 │ ← EXTENSIONS (CLICK HERE!) ⭐
│     │
│  ⚙️  │ ← Settings (at bottom)
└─────┘
```

**Click the 📦 icon** (4th from top)

---

### Method 2: Keyboard Shortcut
Press: **`Ctrl + Shift + X`**

This works from anywhere in the IDE!

---

### Method 3: Command Palette
1. Press **`Ctrl + Shift + P`** (or `Ctrl + P`)
2. Type "Extensions"
3. Select "Extensions: Show Extensions Panel"

## 🚀 First Time Setup

### Step-by-Step for Complete Beginners

1. **Launch EDITH IDE**
   - Double-click the EDITH icon
   - Wait for application to load

2. **Look at the LEFT SIDE**
   - You'll see a vertical bar with icons
   - This is called the "Activity Bar"

3. **Find the 📦 Icon**
   - It's the 4th icon from the top
   - It looks like a grid or package
   - Hover over it and you'll see "Extensions" tooltip

4. **Click It!**
   - The Extensions panel will slide open
   - You'll see "EXTENSIONS" at the top

5. **You're In!**
   - Now you can search for extensions
   - Try typing "python" in the search box

## 🔍 What Should I See?

### When Extensions Panel Opens

**Top Section - SEARCH:**
```
[Search Open VSX Registry...     ] 🔍
```
- This is where you type extension names

**Middle Section - INSTALLED:**
```
Installed (0)
No extensions installed yet
```
- Shows extensions you've installed
- Empty at first (that's normal!)

**Bottom Section - RECOMMENDED:**
```
Recommended                     [↻]
[Extension cards will appear here]
```
- Popular extensions from Open VSX
- Takes 1-2 seconds to load

## ❓ Not Seeing It?

### Troubleshooting

**Problem: Activity Bar is Hidden**
- Press `Ctrl + B` to show/hide sidebar
- The entire left panel should appear

**Problem: Wrong Panel is Open**
- Click the 📦 icon again
- Or try keyboard shortcut: `Ctrl + Shift + X`

**Problem: No Icons in Activity Bar**
- The activity bar is always visible on the far left
- Check if EDITH is running in fullscreen
- Try pressing `F11` to exit fullscreen

**Problem: Application Won't Start**
- Make sure backend is running
- Check if port 8001 is available
- Look for error messages
